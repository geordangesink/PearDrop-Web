PearDrop Support

Need help with installs, invite links, or web downloads? We can help.

Contact Support

Email: support@realpainpoints.com

For faster support, include your device, OS version, browser, and a short description of what happened.

Common Issues

  • Invite link does not open in app: Ensure the latest PearDrop app is installed, then try opening the link again.
  • Web transfer does not start: Check that both peers are online and disable strict content blockers for the session page.
  • Download is blocked by browser: Allow downloads for peardrop.online in your browser site settings.
  • Mobile install issue: Use the latest download links at Downloads.

Before Contacting Us

  • Retry with the latest app/web client version.
  • Test once on another browser or device if possible.
  • Note the exact error message and when it appears.
